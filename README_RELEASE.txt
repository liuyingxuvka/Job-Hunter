Jobflow Desktop Release Package
===============================

1) Double-click START_JOBFLOW_DESKTOP.cmd to launch the app.
2) You can provide API settings either inside the app UI or by environment variables:
   - OPENAI_API_KEY
   - OPENAI_BASE_URL (optional)
   - JOBFLOW_OPENAI_MODEL (optional)
   - AZURE_OPENAI_API_KEY / AZURE_OPENAI_ENDPOINT / AZURE_OPENAI_DEPLOYMENT (optional)

Notes:
- This package excludes tests and smoke scripts.
- Runtime logs and run outputs start empty in this release package.
